Project Overview

This project is a web-based user management and quiz generation system built using Django. It integrates user registration and authentication with AI-powered automatic quiz generation from uploaded PDF documents.

Key Features

  • • User Registration and Authentication: Users register with personal details and profile image. Admin approval is required to activate accounts. Supports secure login and password reset via email OTP.
  • • Admin Dashboard: Admin can log in, view all users, and manage account statuses (activate, deactivate, delete).
  • • AI-Powered Quiz Generation: Users upload PDF files, which are processed to extract text. Using LangChain and Gemini LLM, multiple-choice questions (MCQs) are generated automatically based on the PDF content.
  • • Quiz Interface and Result Evaluation: Users can attempt generated MCQs and receive detailed scoring and feedback.

Technologies Used

  • • Backend: Django and LangChain for web framework, user sessions, and file handling.
  • • File Handling: Django storage for file upload and management.
  • • PDF Processing: PyMuPDF (fitz) for extracting text from PDFs.
  • • AI Integration: Gemini LLM for quiz question generation.
  • • Email: Django email backend for OTP password reset.
  • • Frontend: Django templates with Tailwind CSS for responsive and interactive UI.

Summary

This project combines user management with advanced AI capabilities to deliver a dynamic quiz generation platform. It allows users to create customized quizzes from any educational PDF content, making learning interactive and accessible. Admin controls ensure secure access, and password reset enhances account security.
